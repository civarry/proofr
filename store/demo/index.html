Proofr demo

Writing Playground

A clean page for testing the extension and capturing screenshots. The text below is intentionally rough, so you can fix it in a click.

1. Select some text
2. Right-click → Proofr
3. Pick Proofread, a tone, or Translate

We was hoping to recieve the report by friday, but it dont arrive yet. Their going to send it tomorow instead, so we should of planned for a small delay. Please let me know if you has any questions.

Great for the “before → after” hero shot. Proofread keeps the wording and only fixes grammar and spelling.

Translate

Good morning, everyone. Thank you for joining today’s meeting. Let’s start with a quick update on the project.

Pick “Translate,” then choose a target language.
