# Proofr — Private AI Proofreader & Rewriter

Private AI that proofreads, rewrites in any tone, and translates. Works on your device, no account needed.

Proofr is a Chrome extension (Manifest V3). By default it uses Chrome's Built-in AI (Gemini Nano), which runs entirely on your computer, so your text never leaves your device. You can optionally add a Groq API key to use a cloud model instead.

## Features

- **Proofread** — fix grammar and spelling while keeping your voice
- **Rewrite** — professional, friendly, short, simple, LinkedIn, marketing, executive, academic, and news tones
- **Translate** — English, Spanish, French, German, Japanese
- Works inline on the page (right-click or the popup) with in-place replacement in editable fields
- On-device by default; optional Groq engine

## Privacy

Proofr does not run any servers, collect data, or track you. See the [privacy policy](https://civarry.github.io/proofr/privacy.html).

## Install (development)

1. Clone this repo
2. Open `chrome://extensions`, enable **Developer mode**
3. Click **Load unpacked** and select this folder
